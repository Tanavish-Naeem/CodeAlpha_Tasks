# CodeAlpha Tasks Project

## About
This project contains the solutions for the tasks assigned by CodeAlpha.  
It is made using HTML, CSS, JavaScript.  
The project demonstrates my work on basic IT and programming tasks.

## Tasks
1. **Image Gallery** – A simple gallery to display images dynamically.  
2. **Calculator** – A functional calculator that performs basic arithmetic operations.  
3. **Portfolio** – A personal portfolio website showcasing my projects and skills.

## How to Run
1. Clone the repository:
git clone https://github.com/Tanavish-Naeem/CodeAlpha_Tasks.git
2. Open the folder in VS Code.  
3. Run the code files using programming environment:  
   - For HTML/CSS/JS → Open the HTML files in a browser  `  
   - For C++ → Compile and run using your compiler
